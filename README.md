# Izmir-Houses-Rent-Prices-Prediction-Model
A practical approach to predicting rental prices in İzmir’s housing market and how it can benefit real estate professionals and renters alike. By using data mining techniques, the study identifies key factors—like neighborhood characteristics and socioeconomic indicators— that influence rental costs.
